# 水平居中

## 常规元素水平居中显示

1. 行内元素、不定宽块级元素水平居中

```html
<!-- css -->
<style>
  .outer {
    text-align: center;
  }
</style>
<!-- HTML -->
<div class="outer">
  <span class="inner">水平居中元素</span>
</div>
```

  <!--  more -->

2. 定宽块级元素水平居中：margin

```html
<!-- css -->
<style>
  .inner {
    width: 200px;
    margin: 0 auto;
  }
</style>
<!-- HTML -->
<div class="outer">
  <div class="inner">水平居中元素</div>
</div>
```

3. 图像元素水平居中：clear+display+margin

```html
<!-- css -->
<style>
    img {
        clear: both;
        display: block;
        margin: auto;
    }
</style>
<!-- HTML -->
<div class='outer'>
    <img src="one.png" >水平居中元素</div>
</div>
```

## 浮动元素水平居中

1. 定宽元素

```html
<!-- css -->
<style>
  .inner {
    float: left;
    position: relative;
    left: 50%;
    margin-left: -100px;
    width: 200px;
  }
</style>
<!-- HTML -->
<div class="outer">
  <div class="inner">水平居中的元素</div>
</div>
```

2. 不定宽元素

```html
<!-- css -->
<style>
  .outer {
    float: left;
    position: relative;
    left: 50%;
  }

  .inner {
    float: left;
    position: relative;
    right: 50%; //left: -50%;
  }
</style>
<!-- HTML -->
<div class="outer">
  <div class="inner">水平居中的元素</div>
</div>
```

## 绝对定位元素水平居中

1. 定宽元素

```html
<!-- css -->
<style>
  .outer {
    position: relative;
  }

  .inner {
    width: 200px;
    position: absolute;
    left: 50%;
    margin-left: -100px;
    /*
        left: 0;
        right: 0;
        margin: 0 auto;
        */
  }
</style>
<!-- HTML -->
<div class="outer">
  <div class="inner">水平居中的元素</div>
</div>
```

2. 不定宽元素

```html
<!-- css -->
<style>
  .outer {
    position: relative;
  }

  .inner {
    position: absolute;
    left: 50%;
    transform: translateX(-50%);
  }
</style>
<!-- HTML -->
<div class="outer">
  <div class="inner">水平居中的元素</div>
</div>
```

## 通用方式：flex

```html
<!-- css -->
<style>
  .outer {
    display: flex;
    justify-content: center;
  }
</style>
<!-- HTML -->
<div class="outer">
  <div class="inner">水平居中元素</div>
</div>
```

# 垂直居中

## 常规元素垂直居中

1. 行内元素：line-height

```html
<!-- css -->
<style>
  .outer {
      height： 40px;
  }

  .inner {
      line-height: 40px;
  }
</style>
<!-- HTML -->
<div class="outer">
  <span class="inner">垂直居中元素</span>
</div>
```

2. 通用（行、块、图片）

```html
<!-- css -->
<style>
  .outer {
    display: table-cell;
    vertical-align: middle;
  }
</style>
<!-- HTML -->
<div class="outer">
  <div class="inner">垂直居中元素</div>
</div>
```

## 浮动元素垂直居中

- 定高元素

```html
<!-- css -->
<style>
  .outer {
    height: 200px;
    position: relative;
  }

  .inner {
    float: left;
    height: 100px;
    position: absolute;
    top: 0;
    bottom: 0;
    margin: auto;
    /*
    top: 50%;
    margin-top: -50%;
    */
  }
</style>
<!-- HTML -->
<div class="outer">
  <div class="inner">垂直居中元素</div>
</div>
```

## 绝对定位元素垂直居中

```html
<!-- css -->
<style>
  .outer {
    height: 200px;
    position: relative;
  }

  .inner {
    position: absolute;
    top: 50%;
    transform: translateY(-50%);
  }
</style>
<!-- HTML -->
<div class="outer">
  <div class="inner">垂直居中元素</div>
</div>
```

## 通用

```html
<!-- css -->
<style>
  .outer {
    display: flex;
    align-items: center;
  }
</style>
<!-- HTML -->
<div class="outer">
  <div class="inner">垂直居中元素</div>
</div>
```

# 水平垂直居中

## 常规元素水平垂直居中

- 通用（一列或多列）

```html
<!-- css -->
<style>
  .outer {
    height: 200px; // 效果查看
    width: 300px; // 效果查看
    display: table-cell;
    vertical-align: middle;
    text-align: center;
  }
</style>
<!-- HTML -->
<div class="outer">
  <div class="inner">水平居中元素</div>
</div>
```

## 绝对定位元素水平垂直居中

```html
<!-- css -->
<style>
  .outer {
    height: 200px; // 效果查看
    width: 300px; // 效果查看
    position: relative;
  }

  .inner {
    position: absolute;
    top: 50%;
    left: 50%;
    transform: translate(-50%, -50%);
  }
</style>
<!-- HTML -->
<div class="outer">
  <div class="inner">水平居中元素</div>
</div>
```

## 通用

```html
<!-- css -->
<style>
  .outer {
    height: 200px; // 效果查看
    width: 300px; // 效果查看
    display: flex;
    justify-content: center;
    align-items: center;
  }

  .inner {
    display: inline-block; // 多列
  }
</style>
<!-- HTML -->
<div class="outer">
  <div class="inner">水平居中元素</div>
  <div class="inner">水平居中元素</div>
  <div class="inner">水平居中元素</div>
</div>
```
